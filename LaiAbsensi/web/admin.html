Ikuti langkah mudah berikut untuk mengimpor data ke Google Sheets:

    Buka sheets.google.com dan buat spreadsheet baru
  3. 3
    Pilih File → Import → Upload lalu pilih file CSV yang baru diunduh
  4. 4
    Pilih "Replace spreadsheet" dan klik Import data
